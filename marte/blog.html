Como Adicionar um Novo Post

Para criar uma nova postagem, basta copiar e colar o bloco de código "MODELO DE POSTAGEM" abaixo deste card de instruções. Altere o conteúdo dentro das tags para formatar seu texto.

  • <h2>...</h2> para o título principal (equivale a # em Markdown).
  • <h3>...</h3> para subtítulos (equivale a ## em Markdown).
  • <p>...</p> para parágrafos.
  • <strong>...</strong> para negrito (equivale a **texto**).
  • <em>...</em> para itálico (equivale a *texto*).
  • <blockquote>...</blockquote> para citações (equivale a > texto).
  • <ul><li>...</li></ul> para listas (equivale a - item).
